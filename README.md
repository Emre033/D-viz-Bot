# Discord Döviz Botu 💸

## 🎊 Nasıl Kullanılır
1. `npm i` ve `node .` komutlarıyla botu başlatın.
2. Botu sunucunuza ekleyin.

## 🧨Destek
Eğer problem yaşıyorsan [Discord Sunucumuza](https://discord.gg/akparti) katılabilirsin.
